About The Project

Safire is written with modern frameworks and highly opinionated style choices.

It is a rewrite of Ember in the Sapphire framework.

Getting Started

To get a local copy up and running follow these simple steps.

Prerequisites
  • npm

Installation

  1. Clone the repo

    git clone https://gitlab.com/safire-project/discord-bot/safire.git

  2. Install NPM packages

    NODE_ENV=development npm install

Usage

The following can be used to run Safire, in order of ease of use and reliability. Using Docker will provide reproducible builds due to determinate configuration.

Before doing any of them populate a .env with at least DISCORD_TOKEN=xxxx.... All configuration options can be shown in the example .env

Docker Compose

 
docker-compose up

Docker

 
docker build -t safire .
docker run safire

PM2

 
npx run pm2
npx run pm2stop

Node

 
npm start

For further examples and explanations, please refer to the Documentation
